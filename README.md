## Features

| Feature | Description |
| :--- | :--- |
| **IP Scan** | Detect all devices on your network and monitor for suspicious activity in real-time. |
| **IP Block** | Granular control to automatically block specific malicious or unwanted IP addresses (Single/All). |
| **ARP Spoofing Shield** | Advanced protection layer to prevent Man-in-the-Middle (MITM) attacks and network spoofing. |
| **User-Friendly UI** | Simple, clean, and intuitive interface built with modern Android standards. |

## Compatibility

* **Architecture:** Support **ALL** (armeabi-v7a, arm64-v8a, etc.).
* **Network:** Works flawlessly on most Wi-Fi networks.

## Installation

1. Download the latest **Netra APK** from the [Releases](https://github.com/USERNAME_MAS/NAMA_REPO/releases) section.
